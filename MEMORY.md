## Правило загрузки старых логов

Старые дневные логи из `logs/daily/` не загружаются автоматически. Они читаются только если:

- нужно восстановить контекст конкретной темы;
- студент просит вспомнить прошлую сессию;
- есть противоречие в прогрессе;
- запускается dreaming;
- нужно восстановить статус домашней работы;
- текущая задача явно связана с тем прошлым логом.
